Detailed Description

Utilties for dynamic memory allocation

Wrapper for memory allocation Max/MSP SDK suggests using the Max/MSP "sysmem_" - routines instead of malloc/calloc/free So for Max/MSP define the Preprocessor macro "MAXMSPSDK".
